## How to start

### First

  1. install json-server
     1. npm install -g json-server
     2. yarn global add json-server

  2. start json-server
     1. npx json-server server.json -p 3333

### Starting the application web

1. npm install
2. npm start

or

1. yarn
2. yarn start
